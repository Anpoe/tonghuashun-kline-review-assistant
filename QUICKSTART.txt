K线复盘助手 1.1.1
===================

普通用户推荐使用 GitHub Releases 中的安装版。
从仓库下载源码时，双击 start_recorder.bat 即可运行当前最新版；缺少依赖时会自动准备 .venv。

首次启动：
1. 程序会自动查找同花顺远航版；找不到时请选择 happ.exe。
2. 选择复盘笔记的保存文件夹，可以是 Obsidian 仓库中的目录。
3. 点击“保存并开始”。

使用时打开“新K线训练营”并进入 K 线训练即可，后续截图、拼接和保存均自动完成。
悬浮窗底部可以打开数据看板、保存目录或重新设置。
数据看板会在浏览器中显示收益表现和全部历史记录，数据只从本机复盘目录读取。

设置文件：%APPDATA%\KlineReviewAssistant\config.yaml
程序不会上传截图或交易记录。

本地更新：
- 拉取或替换源码后再次运行 start_recorder.bat，不需要重新安装。
- GitHub 发布包只提供安装版；仓库源码本身就是免安装版本。

注意：
- 需要 Windows 10/11 64 位。
- 需要安装同花顺远航版。
- 首次启动 OCR 组件可能需要数秒初始化。
- 本程序尚未进行代码签名，Windows 可能显示 SmartScreen 提醒。
