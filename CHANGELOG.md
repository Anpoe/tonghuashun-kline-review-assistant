# Changelog

## 未发布

### 新增

- WebUI 训练详情支持删除记录，并同步刷新统计、曲线和历史列表。
- 删除 Markdown 时清理该记录的专属截图，同时保留被其他记录共享的图片。
- 悬浮窗支持创建彩色标签预设，并为当前或下一局训练选择多个标签。
- 标签随复盘笔记保存，WebUI 支持标签展示、文字搜索和下拉筛选。
- 标签选择改为贴近悬浮窗的折叠面板，新建区按需展开，并以勾选、彩色边框和高亮背景明确区分选中状态。
- 仓库根目录新增轻量启动 EXE，拉取源码后可直接双击运行，并在首次启动时自动准备本地 Python 环境。

### 改进

- 缓存训练营蓝框位置，避免每一帧都扫描整个多屏桌面。
- 训练状态改用高频局部 OCR，训练过程中停用无意义的整窗 OCR，首页整页识别降为低频兜底。
- 页面切换与悬浮窗状态立即同步；结果页首次识别后锁定截图，快速点击下一局也不会丢失本局结果。

## 1.1.1 - 2026-07-19

### 修复

- 修复漏识别 `30/30` 后，结果页已经出现但悬浮窗仍停留在等待状态的问题。
- 对底部结算计数器、顶部行情和结果卡片增加局部放大 OCR。
- 结果页至少稳定 1 秒后再读取，降低动画或未完成渲染造成的漏识别。
- 未识别到 `30/30` 时暂存首次有效盘面，结算后自动补录；无法补录时明确显示错误，不再静默等待。

### 工程维护

- 本地启动脚本改为始终运行当前源码，更新后不再需要重新安装。
- 缺少依赖时可自动创建项目内 `.venv`，并拆分运行依赖与构建依赖。
- GitHub Release 仅生成安装程序和校验文件；仓库源码作为免安装版本使用。
- 发布完成后默认清理 PyInstaller 中间目录，减少项目体积和旧版本混用。

## 1.1.0 - 2026-07-19

### 新增

- 内置本地 WebUI 数据看板，可从悬浮窗直接打开。
- 展示训练局数、平均收益、胜率、最佳单局、收益趋势和结果分布。
- 支持 30 天、90 天和全部记录范围，以及股票搜索和盈亏筛选。
- 历史记录详情可查看完整拼接盘面和训练结果卡片。
- 自动读取已有 Markdown 复盘记录，兼容旧版 `pct` 收益格式。

### 改进

- 看板仅在记录变化时重绘，避免轮询打断搜索和浏览。
- 图片接口限制在配置的复盘目录内，所有数据仅在本机提供。

## 1.0.1 - 2026-07-19

同花顺 K线复盘助手的首个公开版本。

### 功能

- 自动启动或连接同花顺远航版，并跟随“新K线训练营”窗口。
- 自动识别训练开始、结束和返回主菜单状态。
- 保存开始与结束盘面，生成带重复区间提示的复盘合图。
- 识别股票名称、代码、训练区间和本局收益率。
- 自动生成 Obsidian 兼容的 Markdown 复盘笔记。
- 提供首次设置向导、悬浮进度窗口和 OCR 自检。
- 适配 Windows DPI 缩放与训练营窗口移动。

### 注意

- 当前版本仅支持 Windows 10/11 64 位。
- 程序没有购买代码签名证书，Windows SmartScreen 可能显示未知发布者提示。
- 本项目是非官方辅助工具，与同花顺及其关联公司无隶属或授权关系。
