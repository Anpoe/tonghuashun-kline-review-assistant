# Changelog

## 1.0.1 - 2026-07-19

同花顺 K线复盘助手的首个公开版本。

### 功能

- 自动启动或连接同花顺远航版，并跟随“新K线训练营”窗口。
- 自动识别训练开始、结束和返回主菜单状态。
- 保存开始与结束盘面，生成带重复区间提示的复盘合图。
- 识别股票名称、代码、训练区间和本局收益率。
- 自动生成 Obsidian 兼容的 Markdown 复盘笔记。
- 提供首次设置向导、悬浮进度窗口和 OCR 自检。
- 适配 Windows DPI 缩放与训练营窗口移动。

### 下载

- `KlineReviewAssistant-Setup-1.0.1.exe`：推荐，带安装和卸载功能。
- `KlineReviewAssistant-1.0.1-portable.zip`：免安装便携版。
- `SHA256SUMS.txt`：安装包和便携版的 SHA-256 校验值。

### 注意

- 当前版本仅支持 Windows 10/11 64 位。
- 程序没有购买代码签名证书，Windows SmartScreen 可能显示未知发布者提示。
- 本项目是非官方辅助工具，与同花顺及其关联公司无隶属或授权关系。
